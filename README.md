# 🏦 Banking System Database (SQL Project)

This project is a simple **Banking System database** built using SQL.  
It simulates basic banking operations such as managing customers, accounts, and transactions.


## 📌 Project Overview

The system is designed to demonstrate how a real-world banking database works at a basic level.
It includes:
- Customer management  
- Bank accounts (Savings / Current)  
- Transaction tracking (Deposit, Withdraw, Transfer)

## ⚙️ Key Features

- Create and manage customers  
- Open bank accounts  
- Perform transactions (deposit, withdraw)  
- Track account balance changes  
- View transaction history  


## 📊 Example Queries

- JOIN between Customers and Accounts  
- GROUP BY to calculate total transactions  
- SELECT queries for account history
